# Enterprise-Managment

This work consists of a program in the C language that manages the employees, departments, and contracts of the company "Furniture for Everyone". In employee management, the user can create, edit, and delete an employee and can also view a list of employee contacts. 
Next, in department management, the user can also create, edit, and delete departments. Finally, in contract management, the user can assign an employee to a department, saving the start and end dates of the contract, export a file with active contracts and another with inactive contracts, and finally, can view a list of contracts about to end and another with employees of a certain department.
